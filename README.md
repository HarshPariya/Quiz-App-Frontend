# Quiz App Frontend

Welcome to the Quiz App Frontend! This project is a React-based web application that allows users to take interactive quizzes.

## 🚀 Live Demo

You can experience the live application here: **[Quiz App Live Demo](https://quiz-app-by-harsh.netlify.app/)**

## 🛠️ Built With

- [React](https://reactjs.org/) - A JavaScript library for building user interfaces.
- Created using `create-react-app`.

## ⚙️ Running the Project Locally

To get a local copy up and running, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HarshPariya/Quiz-App-Frontend.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Quiz-App-Frontend
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will reload if you make edits.
